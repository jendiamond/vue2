2. Listen to the "keydown" event

- Store the value in a data property

(hint: event.target.value gives you the value)

{{ val1 }}

<input type="text"
  v-on:keydown=
  'storeValue1'>
  <p>{ { val1 } }</p>

3. Listen to the "keydown" event

- Store the value in a data property

- Only fire if the "key down" is the ENTER key

{{ val2 }}

<input type="text"
  v-on:keydown.enter=
  'storeValue2'>
  <p>{ { val2 } }</p>
new Vue({
  el: '#exercise',
  data: {
    val1: '',
    val2: '',
  },
  methods: {
    showAlert: function(event) {
      document.getElementById('horn').play();
      alert('ALERT! :O');
    },
    storeValue1: function (event {
      this.val1 = event.target.value;
    },
    storeValue2: function(event) {
      this.val2 = event.target.value;
    },
  }
});
