4. Print the following object (only the values)
    and Create a nested loop for the array:
    

testData: {name: 'TESTOBJECT', id: 10, data: [1.67, 1.33, 0.98, 2.21]}

    (hint: use v-for and v-if to achieve this)


          <ul>
            <li v-for='value in testData'>
              <template v-if='Array.isArray(value)'>
                <div v-for='element in value'>{ { element } }</div>
              </template>
              <template v-else>{ { value } }</template>
            </li>
          </ul>
